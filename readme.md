# How to run the project

1. Download the repository

```bash
git clone git@github.com:SantiagoVargasVe/Ztest.git
```

2. Navigate to the folder

```bash
cd Ztest
```

3. Install dependencies

```bash
npm i
```

4. Run the app

```bash
npm run dev
```

In case you want to run it in production mode

- Build the project

```bash
npm run build
```

- Serve the bundles

```bash
npm run preview
```
